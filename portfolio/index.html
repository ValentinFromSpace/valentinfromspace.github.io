Code

Public GitHub repositories.

  • Middleman
    A completely type-safe way of handling the x-callback-url scheme in Swift.
  • Honey
    A Swift API for interacting with Bear's x-callback-url scheme, based on Middleman. I'm using this myself to automate my journaling practice in the Bear app.
  • Chef
    AKA Chef Bezos, a Discord bot I created for a private server of friends. In two weeks I learned the JS/TS ecosystem, set up VS Code and linters, added typings for a third-party library, figured out Heroku hosting and developed a full-fledged music bot, Blackjack and other gambling minigames, a micro-economy and a role-playing restaurant with an externally extensible menu via AirTable's API and ordering and billing capabilities.
