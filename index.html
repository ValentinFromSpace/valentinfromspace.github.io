Internet Computer

This website is running on the internet computer. The IC, developed by Dfinity, is a web of globally distributed data centers that execute "canisters". A canister is a more capable smart-contract. Canisters include both code and persistent state. You're visiting this website through the friendly domain valwal.com which is pointing at the canister identifier hosting the files.
